# Beginner SQL Portfolio

This repository is a collection of beginner SQL practice projects focused on business-style questions and clear, readable queries.

The goal is to show a foundation in:
- filtering and sorting data
- aggregations and grouped analysis
- joins
- case statements
- common table expressions (CTEs)
- answering practical business questions with SQL

## Files

- `01_select_where_orderby.sql` — basic filtering and sorting
- `02_group_by_having.sql` — grouped metrics and summary analysis
- `03_joins.sql` — combining data across tables
- `04_case_statements.sql` — categorizing and labeling data
- `05_subqueries_ctes.sql` — breaking more complex logic into steps
- `06_business_questions.sql` — example business questions using SQL

## Notes

These examples are built around simple mock business scenarios like orders, customers, and products.
The focus is on writing SQL that is easy to follow and useful for analysis.
